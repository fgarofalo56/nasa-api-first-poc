# dab — Microsoft Data API Builder

Auto-generates REST + GraphQL + OpenAPI over Postgres from `dab-config.json`
(entities: `Material`, `Vendor`, `PurchaseOrder`, `SupplyRisk`; read-only; OData query options).

> [!IMPORTANT]
> Reachable **ONLY** by Kong (internal network) — this is what makes zero-move real.

> [!NOTE]
> Build per PRP §6/§8 Phase 2.
