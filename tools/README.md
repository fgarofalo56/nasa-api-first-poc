# tools
`azure_pricing.py` — queries the public Azure Retail Prices API (no auth) and prints a
dated, source-noted table. Every figure carries: "Source: Azure Retail Prices API, list
price (PAYG), <region>, retrieved <YYYY-MM-DD>; excludes EA/MCA/commit discounts."
No staffing/services dollar figures. Build per PRP §3/§9.
